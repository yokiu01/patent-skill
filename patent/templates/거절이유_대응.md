# 거절이유통지 대응 가이드

특허 심사 과정에서 거절이유통지를 받았을 때 대응하는 방법입니다.

---

## 거절이유 유형 및 대응 전략

### 1. 신규성 위반 (제29조 제1항)

**의미**: 선행기술과 동일한 발명

**심사관 표현 예시**:
> "청구항 1의 발명은 인용발명 1과 동일하므로 신규성이 인정되지 않습니다."

**대응 전략**:

| 방법 | 설명 | 난이도 |
|------|------|:------:|
| **청구항 보정** | 선행기술과 다른 구성요소 추가/한정 | ⭐⭐ |
| **의견서 주장** | 구성요소의 차이점 논증 | ⭐⭐⭐ |
| **삭제** | 해당 청구항 삭제 | ⭐ |

**의견서 작성 예시**:
```
인용발명 1은 [A 구성]을 개시하고 있으나,
본원 발명은 [A' 구성]을 포함하고 있어
양 발명은 [차이점]에서 상이합니다.

구체적으로, 본원 발명의 [A' 구성]은
[기술적 효과]를 달성하기 위한 것으로,
이는 인용발명 1에서는 찾아볼 수 없습니다.
```

---

### 2. 진보성 위반 (제29조 제2항)

**의미**: 선행기술로부터 쉽게 도출 가능

**심사관 표현 예시**:
> "청구항 1의 발명은 인용발명 1, 2의 결합으로 용이하게 도출됩니다."

**대응 전략**:

| 방법 | 설명 | 성공률 |
|------|------|:------:|
| **결합 곤란성 주장** | 인용발명들의 결합 동기 부재 | 중간 |
| **예측 불가능한 효과** | 현저한 기술적 효과 입증 | 높음 |
| **청구항 보정** | 차별적 구성요소 추가 | 높음 |

**의견서 작성 예시**:
```
1. 결합의 곤란성
인용발명 1은 [분야 A]에 관한 것이고,
인용발명 2는 [분야 B]에 관한 것으로,
양 발명을 결합할 동기나 암시가 없습니다.

2. 현저한 효과
본원 발명은 [구체적 수치]의 효과를 달성하며,
이는 인용발명들의 단순 결합으로는 예측 불가능합니다.

3. 보정에 의한 차별화
청구항 1을 보정하여 [차별적 구성]을 명확히 하였습니다.
```

---

### 3. 기재불비 (제42조 제3항, 제4항)

**의미**: 명세서 기재가 불충분

**유형별 대응**:

#### 3-1. 실시 가능 요건 위반 (제42조 제3항 제1호)

**심사관 표현**:
> "통상의 기술자가 명세서 기재만으로 발명을 실시할 수 없습니다."

**대응**:
```
명세서 [단락 00XX]에 기재된 바와 같이,
[구체적 실시 방법]이 개시되어 있으며,
통상의 기술자는 이를 참조하여 발명을 실시할 수 있습니다.

[필요시 보정]
명세서를 보정하여 [추가 설명]을 기재하였습니다.
```

#### 3-2. 청구항 명확성 위반 (제42조 제4항 제2호)

**심사관 표현**:
> "청구항의 '효과적인', '적절한' 등의 표현이 불명확합니다."

**대응**:
- 모호한 표현을 구체적 수치나 명확한 용어로 보정
- 예: "적절한 온도" → "50~100℃의 온도"

#### 3-3. 청구항 뒷받침 위반 (제42조 제4항 제1호)

**심사관 표현**:
> "청구항의 [A 구성]이 명세서에 기재되어 있지 않습니다."

**대응**:
- 명세서에 해당 내용 보정 추가
- 또는 청구항에서 해당 구성 삭제/수정

---

### 4. 단일성 위반 (제45조)

**의미**: 청구항들이 하나의 발명 개념으로 묶이지 않음

**대응**:
1. 분할출원으로 분리
2. 청구항 일부 삭제
3. 기술적 관련성 논증

---

## 보정서 작성 요령

### 보정 가능 범위

| 시기 | 보정 범위 |
|------|-----------|
| 최초 거절이유통지 후 | 최초 명세서 범위 내 자유 보정 |
| 최후 거절이유통지 후 | 청구항 삭제/감축만 가능 |
| 재심사 청구 시 | 청구항 삭제/감축만 가능 |

### 보정서 형식

```
【서류명】 보정서

【제출일자】 20XX년 XX월 XX일

【사건의 표시】
  【출원번호】 10-20XX-XXXXXXX

【보정 대상 서류명】 명세서 또는 청구범위

【보정 내용】

1. 청구항 1을 다음과 같이 보정합니다.

(보정 전)
[기존 청구항 내용]

(보정 후)
[보정된 청구항 내용]

2. 명세서 단락 [00XX]를 다음과 같이 보정합니다.

(보정 전)
[기존 내용]

(보정 후)
[보정된 내용]
```

## 의견서 작성 템플릿

```
【서류명】 의견서

【제출일자】 20XX년 XX월 XX일

【사건의 표시】
  【출원번호】 10-20XX-XXXXXXX

【의견 내용】

1. 거절이유의 요지

심사관님께서는 [거절이유 요약]의 이유로
거절이유를 통지하셨습니다.

2. 본 출원의 발명

본 출원의 발명은 [발명의 핵심] 에 관한 것으로,
[기술적 특징]을 특징으로 합니다.

3. 인용발명과의 대비

3-1. 인용발명 1의 개시 내용
[인용발명 1 요약]

3-2. 본원 발명과의 차이점
[차이점 설명]

4. 진보성의 인정

상기 차이점으로 인해 본원 발명은
[기술적 효과]를 달성하며,
이는 인용발명으로부터 용이하게 도출될 수 없습니다.

5. 결론

따라서, 본원 발명은 신규성 및 진보성이 인정되므로,
특허 등록이 되어야 할 것입니다.
```

---

## 대응 시 주의사항

1. **기한 준수**: 통지서 발송일로부터 2개월 내 대응 (연장 가능)
2. **신규사항 추가 금지**: 최초 명세서 범위 초과 보정 불가
3. **청구항 확장 주의**: 보정으로 청구 범위가 넓어지면 안됨
4. **일관성 유지**: 의견서 주장과 보정 내용 일치
