# 요약서 작성 템플릿

## 요약서란?

요약서는 발명의 핵심 내용을 **한 페이지 이내**로 요약한 문서입니다.
특허 검색 및 기술 정보 제공 목적으로 사용됩니다.

**주의**: 요약서는 청구범위 해석에 사용되지 않습니다.

---

## 기본 구조

```
【요약】
[발명의 목적, 구성, 효과를 간략히 기술 - 400자 내외]

【대표도】
도 [N]
```

---

## 작성 가이드

### 구성 요소

1. **목적** (1~2문장)
   - 본 발명이 해결하려는 과제

2. **구성** (2~3문장)
   - 발명의 핵심 구성요소
   - 청구항 1(독립항)의 요약

3. **효과** (1~2문장)
   - 발명의 주요 효과

---

## 작성 예시

### 예시 1: AI 실행 계획 시스템

```
【요약】
본 발명은 사용자의 행동 패턴을 분석하여 최적의 실행 계획을
자동 생성하는 시스템에 관한 것이다.
본 발명에 따른 시스템은 사용자의 행동 패턴 데이터를 수집하는
데이터 수집부, 최적 실행 시점을 예측하는 분석부, 행동과학 이론에
기반하여 실행 계획을 생성하는 계획 생성부, 및 실행 계획을 제공하는
출력부를 포함한다.
본 발명에 의하면, 사용자의 실행력을 향상시키고 미루기 행동을
효과적으로 개선할 수 있다.

【대표도】
도 1
```

### 예시 2: 이미지 처리 방법

```
【요약】
본 발명은 딥러닝을 이용한 이미지 분류 방법에 관한 것이다.
본 발명에 따른 방법은 입력 이미지를 수신하는 단계,
컨볼루션 신경망을 통해 특징을 추출하는 단계, 및
추출된 특징을 기반으로 이미지를 분류하는 단계를 포함한다.
본 발명에 의하면, 기존 방법 대비 95% 이상의 분류 정확도를
달성할 수 있다.

【대표도】
도 2
```

---

## 작성 체크리스트

- [ ] 400자 내외인가?
- [ ] 목적-구성-효과 순서로 작성했는가?
- [ ] 핵심 구성요소가 포함되었는가?
- [ ] 대표도 번호를 기재했는가?
- [ ] 상표명, 고유명사가 없는가?

---

## 주의사항

1. **길이 제한**: 너무 길면 안됨 (400~500자 권장)
2. **대표도 선정**: 발명을 가장 잘 나타내는 도면 선택
3. **청구항 미포함**: 요약서는 권리범위 해석에 사용 안됨
4. **검색 용도**: 주요 키워드 포함하여 검색 용이하게
