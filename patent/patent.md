# 특허 출원 AI 도우미

변리사 없이 본인 출원을 위한 특허 작성 지원 시스템입니다.

---

## 핵심 작성 철학 (변리사 실무 기준)

### 명세서 작성의 3대 원칙

| 대상 | 목표 | 전략 |
|-----|------|-----|
| **심사관** | 이해하기 쉽게 | 등록 가능성 ↑ |
| **특허권 범위** | 넓게 | 권리범위 ↑ |
| **경쟁자** | 이해하기 어렵게 | 역설계/회피설계 방지 |

### 표현 변환 규칙 (필수)

| 피해야 할 표현 | 권장 표현 | 이유 |
|-------------|---------|------|
| ~한다 | ~할 수 있다 | 권리범위를 한정하지 않기 위함 |
| 바람직하게는, 특히 | 기재 안함 | "바람직하지 않은 것 아니냐" 문제 방지 |
| 바람직한 실시예 | 일 측면에 따른 일 실시예 | 권리행사시 문제 방지 |
| 25% 향상 (효과란) | 향상된다 | 수치 미달시 권리행사 어려움 |
| x, y, z 좌표 | 도면 참조 또는 정의 추가 | 기준 불명확 방지 |

### 수치/정량적 효과 처리

```
❌ 【발명의 효과】에 "정확도 25% 향상" 기재
✅ 【발명의 효과】에 "정확도가 향상된다" 기재
✅ 【실시예】에 "실험 결과 25% 향상 (72%→90%)" 기재
```

> **이유**: 특허권 행사 범위는 효과가 아닌 과제해결원리(=구성)로 판단. 수치 효과 미달시 권리행사 어려움.

## 작업 흐름

### 1단계: 발명 분석
- 발명의 핵심 기술 파악
- 기술 분야 분류 (IPC 코드 추정)
- 해결하려는 과제 정리

### 2단계: 선행기술 조사
WebSearch 도구로 다음 검색 수행:
- KIPRIS 특허 검색: `site:kipris.or.kr [핵심 키워드]`
- Google Patents: `site:patents.google.com [핵심 키워드]`
- 학술 논문: `[기술 키워드] 논문 OR paper`

> ⚠️ **중요**: 선행기술문헌은 **1개 정도만**, **본 발명과 가장 상이한 것**을 기재.
> 유사한 선행기술을 많이 기재하면 진보성흠결(특허법 제29조 제2항) 거절 근거로 사용됨.

### 3단계: 차별점 분석
- 선행기술 대비 신규성 확인
- 진보성 (기술적 효과) 정리
- 회피 설계 필요 여부 판단

### 4단계: 명세서 작성
templates/명세서.md 양식에 따라 작성:
1. 발명의 명칭
2. 기술분야
3. 배경기술 (선행기술 + 문제점)
4. 해결하려는 과제
5. 과제 해결 수단
6. 발명의 효과
7. 발명을 실시하기 위한 구체적인 내용

### 5단계: 청구항 작성
templates/청구항.md 가이드에 따라 작성:
- 독립항: 발명의 핵심 (필수 구성요소만)
- 종속항: 세부 실시예 (선택적 구성요소)

### 6단계: 요약서 작성
templates/요약서.md 양식에 따라 작성

### 7단계: 파일 출력
작성된 내용을 파일로 저장:
- `[발명명칭]_명세서.md`
- `[발명명칭]_청구항.md`
- `[발명명칭]_요약서.md`

## 한국 특허 출원 기본 정보

### 출원 비용 (2024년 기준)
| 항목 | 비용 |
|------|------|
| 출원료 | 46,000원 |
| 심사청구료 (20청구항 기준) | 143,000원 + 44,000원×항수 |
| 등록료 (1~3년차) | 매년 15,000원 + 13,000원×항수 |

### 출원 절차
1. 명세서/청구항/요약서/도면 작성
2. 특허로(www.patent.go.kr) 전자출원
3. 방식심사 (1~2주)
4. 심사청구 (출원 후 3년 이내)
5. 실체심사 (12~18개월)
6. 등록결정 또는 거절이유통지

### 본인 출원 가능 여부
- **가능**: 한국은 변리사 대리 의무 없음
- 다만 복잡한 특허는 전문가 검토 권장
- 이 도구는 법적 조언이 아닌 작성 보조 도구임

## 주의사항

1. **선행기술 조사 필수**: 출원 전 반드시 유사 특허 확인
2. **청구항이 권리범위**: 청구항 작성이 가장 중요
3. **도면 권장**: 기술 내용을 명확히 하려면 도면 포함
4. **비밀 유지**: 출원 전 발명 내용 공개 금지 (신규성 상실)
5. **출원일 확보**: 빠른 출원일 확보가 중요 (선출원주의)

### 변리사 실무 기준 추가 주의사항

6. **선행기술문헌 최소화**: 1개만 기재, 본 발명과 가장 다른 것 선택
7. **IPC 코드 직접 기재 불필요**: 심사관이 분류함
8. **효과란에 수치 금지**: "25% 향상" → 실시예로 이동
9. **"상기" 사용 주의**: 위에 기재한 것만 참조 가능
10. **1발명 1출원**: 시스템/방법/기록매체 분리 출원 검토
11. **출원인 확인**: 법인은 법인등기 필수, 개인은 주민등록상 이름

## 특허청 공식 참고 자료

| 자료 | 링크 | 용도 |
|------|------|------|
| **특허로** | https://www.patent.go.kr | 전자출원 시스템 |
| **KIPRIS** | https://www.kipris.or.kr | 선행기술 검색 |
| **모범명세서** | 특허로 → 출원/등록안내 → 모범명세서 | 기술분야별 예시 |
| **명세서 양식** | http://dtox.patent.go.kr | 워드 양식 다운로드 |
| **특허청 고시** | https://www.kipo.go.kr | 법령/서식 |

### 선행기술 검색 방법 (KIPRIS)
1. https://www.kipris.or.kr 접속
2. "특허/실용신안" 선택
3. 키워드 검색 (발명의 핵심 기술)
4. IPC 분류로 필터링
5. 유사 특허 출원번호/등록번호 확보
