# 심사관 거절 패턴 분석 가이드

특허 심사에서 자주 발생하는 거절이유 패턴과 대응 전략을 정리한 가이드입니다.

---

## 거절이유 통계 (한국 특허청 기준)

| 거절이유 | 비율 | 조항 |
|----------|------|------|
| **진보성 부정** | ~60% | 제29조 제2항 |
| **신규성 부정** | ~15% | 제29조 제1항 |
| **기재불비** | ~15% | 제42조 제3항, 4항 |
| **청구범위 불명확** | ~8% | 제42조 제4항 |
| **기타** | ~2% | 산업상 이용가능성 등 |

---

## 1. 진보성 거절 패턴

### 패턴 1-1: 단순 결합 논리

**심사관 논리:**
> "선행기술 A의 구성요소 X와 선행기술 B의 구성요소 Y를 결합하면
> 청구항 발명에 용이하게 도달할 수 있음"

**거절문 예시:**
```
인용발명 1(한국공개특허 10-2020-XXXXXXX)에는 영상에서 관절 좌표를
추출하는 포즈 추정부가 개시되어 있고,
인용발명 2(한국등록특허 10-XXXXXXX)에는 LSTM을 이용한 시계열 분석이
개시되어 있으므로, 이들을 결합하여 본 발명에 용이하게 도달할 수 있음.
```

**대응 전략:**

1. **결합 동기 부정**
   - 두 선행기술의 기술분야가 다름을 주장
   - 결합을 시사하는 암시가 없음을 강조
   - "후견적 고찰(hindsight)"임을 지적

2. **결합 저해 요인 제시**
   - 선행기술이 다른 방향을 제시함
   - 결합 시 기술적 모순 발생
   - 당업자가 결합을 회피할 이유

3. **예측 불가능한 효과 주장**
   - 단순 합 이상의 상승효과(시너지)
   - 정량적 효과 데이터 제시
   - 해결된 기술적 과제의 특수성

**의견서 예시:**
```
1. 인용발명들의 결합 동기 부재
인용발명 1은 단순 포즈 추정에 관한 것이고, 인용발명 2는 자연어 처리
분야의 LSTM 적용에 관한 것으로, 두 발명의 기술분야가 상이하여
당업자가 이들을 결합할 동기가 없습니다.

2. 예측 불가능한 효과
본 발명은 Bi-LSTM의 양방향 분석을 통해 동작의 시작점과 종료점을
동시에 고려함으로써, 단방향 분석 대비 [정확도 15% 향상]이라는
예측 불가능한 효과를 달성합니다.
```

### 패턴 1-2: 설계 변경 논리

**심사관 논리:**
> "청구항의 구성은 선행기술의 단순한 설계 변경에 불과함"

**거절문 예시:**
```
청구항에서 'Bi-LSTM'을 사용하는 것은 인용발명의 'LSTM'을 양방향으로
변경한 것에 불과하며, 이는 당업자가 용이하게 선택할 수 있는
설계적 사항임.
```

**대응 전략:**

1. **기술적 의의 강조**
   - 설계 변경이 아닌 기술적 개선임을 주장
   - 변경으로 인한 구체적 효과 제시
   - 해결된 기술적 과제 명시

2. **비자명성 입증**
   - 당업자가 해당 변경을 회피할 이유
   - 선행기술이 다른 방향을 제시
   - 업계의 기술적 편견(technical prejudice)

**의견서 예시:**
```
Bi-LSTM으로의 변경은 단순한 설계 변경이 아닙니다.

1. 기술적 과제의 차이
인용발명의 단방향 LSTM은 과거 데이터만 참조하여 실시간 예측에
적합하나, 본 발명의 Bi-LSTM은 전체 동작 시퀀스를 분석하여
동작 완료 후 정확한 평가를 제공합니다.

2. 응용 분야의 차이
실시간 예측이 아닌 '사후 분석 및 교육'이라는 새로운 응용 분야에
적합한 구성으로, 당업자가 용이하게 도달할 수 없습니다.
```

## 2. 신규성 거절 패턴

### 패턴 2-1: 동일 구성 주장

**심사관 논리:**
> "청구항의 모든 구성요소가 선행기술에 개시되어 있음"

**대응 전략:**

1. **차이점 특정**
   - 선행기술에 없는 구성요소 명시
   - 용어의 차이 설명
   - 기능/효과의 차이 강조

2. **청구항 해석 주장**
   - 청구항 용어의 정의 명확화
   - 명세서 기재에 기반한 해석
   - 선행기술과의 의미 차이

**의견서 템플릿:**
```
심사관님의 인용발명 분석을 검토한 결과, 다음의 차이점이 있습니다:

| 구성 | 본 발명 | 인용발명 | 차이점 |
|------|---------|----------|--------|
| 구성1 | [내용] | [내용] | [차이] |
| 구성2 | [내용] | [내용] | [차이] |
| 구성3 | [내용] | 미개시 | 신규 구성 |

특히, [구성3]은 인용발명에 전혀 개시되어 있지 않으므로,
본 발명은 신규성을 가집니다.
```

## 3. 기재불비 거절 패턴

### 패턴 3-1: 발명의 상세한 설명 불충분

**심사관 지적:**
> "발명의 상세한 설명에 당업자가 발명을 실시할 수 있을 정도로
> 기재되어 있지 않음"

**대응 전략:**

1. **보정으로 대응**
   - 실시예 추가
   - 구체적 수치/파라미터 보충
   - 동작 원리 상세 설명

2. **의견서로 대응**
   - 명세서 내 근거 제시
   - 당업자의 기술 수준 주장
   - 출원일 당시 기술 상식 인용

**흔한 지적 사항:**

| 지적 | 대응 방법 |
|------|-----------|
| 알고리즘 불명확 | 수식/의사코드 추가 |
| 학습 데이터 미기재 | 데이터셋 특성 추가 |
| 파라미터 미기재 | 하이퍼파라미터 범위 추가 |
| 효과 입증 부족 | 실험 결과/비교 데이터 추가 |

---

### 패턴 3-2: 청구항이 발명의 설명에 의해 뒷받침되지 않음

**심사관 지적:**
> "청구항의 권리범위가 발명의 상세한 설명에 의해 뒷받침되지 않음"

**대응 전략:**

1. **청구항 감축 보정**
   - 명세서에 기재된 범위로 한정
   - 종속항 내용을 독립항에 병합

2. **명세서 보정** (신규사항 추가 금지!)
   - 기재된 내용의 명확화만 가능
   - 새로운 실시예 추가 불가

**의견서 예시:**
```
청구항 1의 "신경망"은 명세서 [0023]단락에 기재된 LSTM, Bi-LSTM,
GRU 등을 포괄하는 상위 개념으로서, 명세서에 의해 충분히
뒷받침됩니다.

또한, 당업자라면 명세서의 기재에 기초하여 다른 순환 신경망
구조에도 본 발명을 적용할 수 있음이 자명합니다.
```

## 4. 소프트웨어/AI 특허 특수 거절 패턴

### 패턴 4-1: 자연법칙 이용 여부

**심사관 논리:**
> "청구항의 발명이 자연법칙을 이용한 것인지 불명확함"

**대응 전략:**

1. **기술적 구성 강조**
   - 하드웨어 구성요소 명시
   - 데이터 처리의 기술적 측면 강조
   - "프로세서에 의해 실행되는" 등 추가

2. **기술적 효과 명시**
   - 처리 속도 향상
   - 정확도 개선
   - 메모리 효율화

**보정 예시:**
```
[보정 전]
사용자 데이터를 분석하여 결과를 출력하는 방법

[보정 후]
프로세서가 메모리에 저장된 사용자 데이터를 읽어들이는 단계;
상기 프로세서가 신경망 모델을 이용하여 상기 데이터를 분석하는 단계; 및
상기 프로세서가 분석 결과를 디스플레이부에 출력하는 단계;를 포함하며,
상기 분석 단계에서 연산량을 30% 감소시키는 것을 특징으로 하는
데이터 분석 방법.
```

---

### 패턴 4-2: 공지 기술의 단순 적용

**심사관 논리:**
> "공지의 AI 기술(CNN, LSTM 등)을 특정 분야에 단순 적용한 것에 불과"

**대응 전략:**

1. **기술적 과제의 특수성**
   - 해당 분야 특유의 기술적 문제
   - 공지 기술 적용의 어려움
   - 추가적인 기술적 연구 필요성

2. **맞춤형 구성 강조**
   - 분야 특화 전처리/후처리
   - 파라미터 최적화
   - 데이터 특성에 맞는 모델 수정

**의견서 예시:**
```
본 발명은 단순히 Bi-LSTM을 적용한 것이 아니라:

1. 한국무용의 특수성을 반영한 전처리
   - 33개 관절 중 12개 핵심 관절 선별 (도 3 참조)
   - 한국무용 특유의 곡선 동작 추출 알고리즘

2. 도메인 특화 학습 데이터
   - 5,000개 이상의 전문가 동작 데이터
   - 난이도별 분류 체계 (초급/중급/고급)

3. 예측 불가능한 효과
   - 기존 범용 동작 인식 대비 정확도 25% 향상
```

## 5. 거절 대응 체크리스트

### 거절이유통지 수신 시

- [ ] 거절이유 조항 확인 (29조 1항/2항, 42조 등)
- [ ] 인용 선행기술 입수 및 분석
- [ ] 대응 기한 확인 (2개월, 연장 가능)
- [ ] 대응 전략 수립 (의견서/보정서/분할)

### 의견서 작성 시

- [ ] 심사관 논리의 오류 지적
- [ ] 차이점의 기술적 의의 설명
- [ ] 예측 불가능한 효과 주장
- [ ] 보정 시 신규사항 추가 금지 확인

### 보정서 작성 시

- [ ] 최초 명세서 범위 내 보정인지 확인
- [ ] 청구범위 과도한 감축 피하기
- [ ] 보정 근거(명세서 단락 번호) 명시
- [ ] 종속항 병합 시 논리적 일관성 확인

## 6. 거절 패턴별 성공률

| 거절이유 | 대응 성공률 | 핵심 전략 |
|----------|-------------|-----------|
| 진보성 (결합) | ~40% | 결합 동기 부정 + 효과 주장 |
| 진보성 (설계변경) | ~50% | 기술적 의의 강조 |
| 신규성 | ~60% | 구성 차이 명확화 |
| 기재불비 | ~70% | 보정으로 보완 |
| 청구범위 불명확 | ~80% | 용어 정의 추가 |

## 7. 실전 대응 템플릿

### 진보성 거절 의견서 템플릿

```
【의견서】

1. 거절이유의 요지
심사관님께서는 인용발명 1, 2를 결합하면 본 발명에 용이하게
도달할 수 있다고 지적하셨습니다.

2. 본 출원인의 의견

2-1. 인용발명의 분석
[인용발명 1]: (핵심 내용 요약)
[인용발명 2]: (핵심 내용 요약)

2-2. 본 발명과의 차이점
본 발명의 청구항 1은 다음의 구성을 포함합니다:
(a) ...
(b) ...
(c) ... ← 인용발명에 미개시

2-3. 결합의 곤란성
(1) 기술분야의 차이: ...
(2) 결합 동기의 부재: ...
(3) 결합 저해 요인: ...

2-4. 예측 불가능한 효과
본 발명은 인용발명들의 단순 결합으로는 예측할 수 없는
다음의 효과를 달성합니다:
- 정량적 효과 1: ...
- 정량적 효과 2: ...

3. 결론
이상의 이유로, 본 발명은 인용발명들로부터 용이하게 도출될 수 없으며,
진보성을 가지므로 특허등록되어야 합니다.
```
