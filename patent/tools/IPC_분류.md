# IPC 코드 분류 가이드

IPC(International Patent Classification)는 특허를 기술 분야별로 분류하는 국제 표준입니다.

---

## 주요 섹션 (A~H)

| 섹션 | 분야 | 예시 |
|:----:|------|------|
| **A** | 생활필수품 | 농업, 식품, 의류, 건강 |
| **B** | 처리조작; 운수 | 제조, 인쇄, 운송 |
| **C** | 화학; 야금 | 화학물질, 재료 |
| **D** | 섬유; 제지 | 직물, 종이 |
| **E** | 고정구조물 | 건축, 토목 |
| **F** | 기계공학 | 엔진, 펌프, 무기 |
| **G** | 물리학 | 계측, 광학, **컴퓨터** |
| **H** | 전기 | 전기회로, 통신 |

---

## IT/소프트웨어 특허용 IPC 코드

### G06 - 계산; 계수 (컴퓨터/소프트웨어 핵심)

| IPC | 설명 | 적용 예시 |
|-----|------|-----------|
| **G06F** | 전기적 디지털 데이터 처리 | 일반 소프트웨어, 앱 |
| **G06N** | 특정 계산 모델에 기초한 컴퓨터 시스템 | **AI, 머신러닝, 딥러닝** |
| **G06Q** | 행정, 상업, 경영, 감독 목적 데이터 처리 | 핀테크, 이커머스 |
| **G06T** | 이미지 데이터 처리 또는 생성 | 이미지 처리, 컴퓨터 비전 |
| **G06V** | 이미지 또는 비디오 인식 | 영상 인식, 포즈 추정 |
| **G06K** | 데이터 인식; 데이터 제시 | OCR, 바코드 |

## 기술 분야별 IPC 추천

### AI/머신러닝 서비스

```
주분류: G06N 3/04 (신경망 아키텍처)
부분류: G06N 20/00 (머신러닝)
        G06F 18/00 (패턴인식)
```

### 영상/동작 분석

```
주분류: G06V 40/00 (인체 인식)
부분류: G06V 10/00 (이미지 특징 추출)
        G06T 7/00 (이미지 분석)
        G06N 3/04 (신경망)
```

### 교육/학습 시스템

```
주분류: G09B 5/00 (교육 또는 훈련 장치)
부분류: G06Q 50/20 (교육 관련 서비스)
        G06F 16/00 (정보 검색)
```

### 모바일 앱/서비스

```
주분류: G06F 3/00 (사용자 인터페이스)
부분류: H04L 67/00 (네트워크 서비스)
        G06Q 30/00 (상거래)
```

### 헬스케어/피트니스

```
주분류: A63B 24/00 (운동 훈련 장치)
부분류: G16H 20/00 (건강관리 ICT)
        A61B 5/00 (진단 측정)
```

### 음성/자연어 처리

```
주분류: G10L 15/00 (음성 인식)
부분류: G06F 40/00 (자연어 처리)
        G10L 13/00 (음성 합성)
```

## IPC 검색 방법

### 1. WIPO IPC 검색
- https://www.wipo.int/classifications/ipc/en/
- 키워드로 IPC 코드 검색

### 2. KIPRIS 유사 특허 참조
- 유사 특허의 IPC 코드 확인
- 해당 코드 참고하여 적용

### 3. 특허청 IPC 가이드
- https://www.kipo.go.kr
- IPC 분류 안내서 참조

## 자동 IPC 추천 로직

발명 설명에서 키워드를 추출하여 IPC를 추천합니다:

```
키워드 → IPC 매핑

"딥러닝", "신경망", "AI" → G06N 3/00
"LSTM", "RNN", "CNN" → G06N 3/04
"머신러닝" → G06N 20/00
"이미지 인식" → G06V 10/00
"동작 인식", "포즈" → G06V 40/20
"음성 인식" → G10L 15/00
"교육", "학습" → G09B 5/00
"모바일 앱" → G06F 3/00
"결제", "커머스" → G06Q 30/00
"의료", "진단" → G16H 20/00
```

